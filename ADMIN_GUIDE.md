# 🍅 ポモドーロBGM Bot — 管理者向け導入ガイド

## これは何？

ボイスチャンネルで「ポモドーロタイマー付きのYouTube作業BGM」をボタン操作で再生できるDiscord Botです。

ポモドーロタイマーはYouTube動画側に含まれているため、Botの仕組みはとてもシンプルです。「YouTubeの音声をVCで流すだけ」の軽量な構成になっています。

---

## できること

テキストチャンネルにこのようなUIが表示されます：

```
🍅 ポモドーロBGM

[🌊 波] [🌳 森] [🔥 焚き火] [☕ あつ森] [🌧 雨]
[⏹ 停止] [📊 再生状況]
```

ボイスチャンネルに入った状態でボタンを押すと、選んだBGMがVCで再生されます。全員が同じBGMを聴きながら作業できます。誰もいなくなったら自動で退出します。

## 管理者がやること（全4ステップ・約15分）

### ステップ1：Botを作成する（5分）

1. **[Discord Developer Portal](https://discord.com/developers/applications)** を開く
2. 右上の **「New Application」** をクリック
3. 名前を入力（例：`ポモドーロBGM`）→ Create
4. 左メニューの **「Bot」** をクリック
5. **「Reset Token」** → 表示されたトークンをコピー（後で使います）
6. 下にスクロールして **Privileged Gateway Intents** の以下をONにする：
   - `SERVER MEMBERS INTENT`
   - `MESSAGE CONTENT INTENT`
7. 左メニューの **「General Information」** に戻り、**Application ID** をコピー（後で使います）

### ステップ2：サーバーに招待する（2分）

以下のURLの `YOUR_CLIENT_ID` を、ステップ1でコピーした **Application ID** に置き換えて、ブラウザで開いてください：

```
https://discord.com/api/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=3147776&scope=bot
```

対象のサーバーを選んで「認証」を押せば招待完了です。

必要な権限（URLに含まれています）：

- チャンネルを見る
- 接続（ボイスチャンネル）
- 発言（ボイスチャンネル）
- メッセージを送信
- 埋め込みリンク

### ステップ3：IDをメモする（2分）

Discordアプリの **設定 → 詳細設定 → 開発者モード** をONにしてください。

以下2つのIDをコピーします：

- **サーバーID**：サーバーアイコンを右クリック → 「サーバーIDをコピー」
- **テキストチャンネルID**：BGM操作UIを表示したいチャンネルを右クリック → 「チャンネルIDをコピー」

### ステップ4：デプロイする（5〜10分）

#### Railway（おすすめ）

1. **[Railway](https://railway.app/)** にGitHubアカウントでログイン
2. **「New Project」** → **「Deploy from GitHub repo」**
3. このリポジトリを選択
4. **「Variables」タブ** で以下4つを設定：

| 変数名 | 設定する値 |
|--------|-----------|
| `DISCORD_TOKEN` | ステップ1でコピーしたBotトークン |
| `CLIENT_ID` | ステップ1でコピーしたApplication ID |
| `GUILD_ID` | ステップ3でコピーしたサーバーID |
| `TEXT_CHANNEL_ID` | ステップ3でコピーしたテキストチャンネルID |

5. **「Deploy」** をクリック

テキストチャンネルにBGM選択ボタンが表示されたら成功です。

## セキュリティについて

- Bot権限は最小限（VC接続・発言・メッセージ送信のみ）
- サーバーの設定変更やメンバー管理などの権限は一切不要
- 危険な操作は行いません

## 運用負担について

- 基本的に放置でOKです
- 常駐型ではなく、ユーザーがボタンを押したときだけ動きます
- 誰もいなくなったら自動で退出します
- YouTube再生が稀に止まることがあります → ボタンをもう一度押すだけで復旧します

## 技術構成（参考）

- Node.js + discord.js
- YouTube音声ストリーミング（play-dl）
- Railwayで常時稼働（無料枠で対応可能）

---

## 困ったときは

| 症状 | 対処 |
|------|------|
| ボタンを押しても反応しない | Botがオンラインか確認 → Railwayのログを確認 |
| 音が出ない | VCに入っているか確認 → もう一度ボタンを押す |
| 途中で音が止まる | もう一度BGMボタンを押す（自動リトライも入っています） |
| UIが消えた | Botを再起動すると再表示されます |
